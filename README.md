# amer8 Homebrew Tap

Homebrew formulas and casks for open-source tools by Amer Alimanović.

## Install

```sh
brew tap amer8/tap
brew install amer8/tap/apibconv
brew install --cask amer8/tap/pulsebar
```

## Available Packages

- `apibconv`: Convert between API Blueprint, OpenAPI, and AsyncAPI specifications.
- `pulsebar`: macOS menu bar monitor for NVIDIA DGX Spark telemetry.
